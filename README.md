# QueryBuilder

### creator: Sokolov Anton

### to install:
- you can install using ssh/http github keys
- make sure, your php.ini file includes [pdo extensions](https://www.php.net/manual/ru/book.pdo.php)
  for example:
  ```
  extension=pdo_mysql
  extension=pdo_pgsql
  extension=pdo_sqlite
  ```
